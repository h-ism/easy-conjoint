How to use

1. Design the experiment

On the Design tab, add attributes (e.g. Education, Country of origin, Job) and their levels. Use Load sample study to see a worked example. Weights let you over- or under-sample a level; equal weights give uniform randomization (the standard choice).

2. Constraints

Add constraints to rule out nonsensical combinations — e.g. forbid Job = Doctor together with Education = No formal. Each profile is redrawn until it satisfies all constraints. Avoid over-constraining, which shrinks the profile space and can bias estimates.

3. Run the survey

Go to the Survey tab. Each respondent gets freshly randomized profiles. Enter or auto-generate a respondent ID, complete the tasks, and the response is saved automatically. Share the single conjoint.html file, or host it — everything runs locally in the browser.

4. Export & analyze

The Data tab exports a long-format CSV: one row per profile per task, with a chosen (0/1) outcome column and one column per attribute. This is exactly the shape AMCE estimation expects. In R:

library(cjoint); results <- amce(chosen ~ Education + Job + ..., data=d, cluster=TRUE, respondent.id="respondent_id")

Notes on randomization

Levels are drawn independently for each attribute and profile (fully randomized design). Attribute display order is randomized per respondent but held constant across their tasks. Save your design with Export JSON so it is reproducible and shareable.
